# members
1. Install Homebrew MacOs
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2. Change Brew path
export PATH=/opt/homebrew/bin:$PATH

3. Install Node and Npm
brew install node; brew install npm

4. In Client
npm start

5. In Flask-server
python3 server.py
